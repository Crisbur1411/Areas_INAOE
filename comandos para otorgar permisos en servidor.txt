comandos para otorgar permisos a la carpeta de res/temp

comando para ver los permisos de la carpeta
ls -laZ /var/www/html/liberacion_maina_funcional/res/

comando para otorgar todos los permisos a las carpetas dentro de res/
chmod -R 777 /var/www/html/liberacion_maina_funcional/res/


comando para dejar la carpeta con permisos de escritura para Apache/PHP
sudo chcon -R -t httpd_sys_rw_content_t /var/www/html/liberacion_maina_funcional/res/

se repite el comando para ver los permisos de la carpeta
ls -laZ /var/www/html/liberacion_maina_funcional/res/



con esto se valida que asi eran los permisos al inicio
drwxrwxrwx. usuario usuario unconfined_u:object_r:httpd_sys_content_t:s0 temp

y asi con el comando de escritura para Apache/PHP
drwxrwxrwx. usuario usuario unconfined_u:object_r:httpd_sys_rw_content_t:s0 temp
